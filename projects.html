FIGMA PROTOTYPE MOCKUP

Figma

Project Overview: A prototype mockup of a movie database website created with figma. 12 interactive artboards are created, 4 artboards for desktop view, 4 artboards for table view, and 4 more for mobile view. Using figma could be a critical skill in the situation of collaboration. Understanding UI/UX fundamentals extremely help developers to figure out the whole process of web publishing. Prototyping makes developers to understand which functions they would add to the website easily.

What I've learned: I found out figma makes people can do real-time collaboration and enabling multiple users to work on a design simultaneously. Figma also offers a wide range of plugins to extend functionality, I enjoyed using useful plugins such as iconify, and 66frameworks. I could create all prototype functions I imagined. All tools in figma are very user-friendly. These features make it a versatile tool for individual designers and teams alike, streamlining the design process and facilitating easy sharing and feedback.
